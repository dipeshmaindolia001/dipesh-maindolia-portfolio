Designed with a clean, high-contrast "paper CV" editorial theme (Fraunces serif typography, stone-slate readability, hover highlight blocks, and scroll-activated counting statistics).

- **Project Catalog (`projects.html`)**: Repository logs for three shipped end-to-end analytics pipelines.

## 📂 Shipped Project Profiles

### 1. [YouTube Trending Data Pipeline on AWS](https://github.com/dipeshmaindolia001/YouTube-Trending-Data-Pipeline-on-AWS)
- **Category**: Cloud Data Engineering
- **Stack**: AWS S3, Lambda, Glue, Step Functions, Athena, Python
- **Details**: Implemented a **Medallion Data Architecture** (Bronze landing, Silver cleaning, Gold Parquet partitioning) to ingest trending videos via YouTube Data API and automate distributed schema mappings via AWS Step Functions.

### 2. [Flight Price Prediction — AWS SageMaker](https://github.com/dipeshmaindolia001/Flight-Price-Prediction-AWS-Sagemaker)
- **Category**: Machine Learning Engineering
- **Stack**: AWS SageMaker, XGBoost, Scikit-Learn, Streamlit, Pandas, Python
- **Details**: Built an end-to-end regression estimator using SageMaker containers. Tuned hyperparameters via **Bayesian Optimization (HPO)** and hosted a serverless REST endpoint consumed by an interactive Streamlit UI.

### 3. [Ola Data Analytics — End to End](https://github.com/dipeshmaindolia001/Ola-Data-Analytics-End-to-End-Project)
- **Category**: Business Intelligence & SQL Analytics
- **Stack**: Power BI, SQL, Excel, DAX
- **Details**: Modeled ridebooking transaction schemas to analyze surges, driver ratings, and booking drop-off percentages. Modeled interactive multi-page dashboards mapping booking trends and operational metrics.
